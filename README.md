# BcAuthGuard

BcAuthGuard は、管理画面ログインの試行制限と認証監査ログを提供する baserCMS5 向けプラグインです。

## 初期実装で含む機能

- Admin ログインの失敗回数に応じた一時ロック
- ログイン成功時の失敗カウンタ解放
- ログイン失敗 / ロック開始 / ロック中拒否 / IP拒否の監査ログ保存
- 単体IP / CIDR 指定によるログイン拒否（設定ベース）
- ロック中一覧（検索）と手動解除
- 最近の動き（Dblog）への Guard 固有イベント記録（ロック開始 / ログイン制限解除 / IP拒否 / ロック中拒否）

※ 通常のログイン成功 / ログアウトの監査ログと最近の動き記録は BcAuthCommon 側で処理します。

## デフォルト設定

- 10分間に5回失敗でロック
- ロック時間は10分
- IP拒否は有効

`config/setting_customize.php` を作成すると設定を上書きできます。

### blockedIps 設定例

- 単体IP: `192.0.2.10`
- CIDR: `198.51.100.0/24`
- IPv6 CIDR: `2001:db8::/32`

## 管理画面

- 認証ガード設定
	- `Plugins > 認証ガード設定`
- ロック中一覧
	- `Plugins > ロック中一覧`
	- 検索条件: 状態 / プレフィックス / ログインID / IPアドレス
	- ロック中レコードに対して手動解除を実行可能
